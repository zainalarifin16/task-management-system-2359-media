## Docker
```bash
# running all container
$ docker-compose up

# refresh your container
$ docker-compose up --build -V

```

## On Local

### Installation

```bash
$ npm install
```

### Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

### Test

```bash
# unit tests
$ npm run test

# test coverage
$ npm run test:cov
```
